> [!IMPORTANT]
Azure Load Balancer поддерживает два типа: категории "Базовый" и "Стандартный". В этой статье рассматривается Azure Load Balancer категории "Базовый". Хотя Load Balancer категории "Базовый" используется в режиме общедоступной версии, служба Load Balancer категории "Стандартный" сейчас доступна в режиме общедоступной предварительной версии. См. дополнительные сведения в статье [Обзор Azure Load Balancer уровня "Стандартный" (предварительная версия)](https://aka.ms/AzureLoadBalancerStandard).
