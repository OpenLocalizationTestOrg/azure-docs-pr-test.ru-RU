
Код для всех функций в приложении определенной функции расположен в корневую папку, содержащую файл конфигурации главного узла и один или несколько вложенных папках. Каждая вложенная папка содержит код для отдельной функции, как показано в следующем примере:

```
wwwroot
 | - host.json
 | - mynodefunction
 | | - function.json
 | | - index.js
 | | - node_modules
 | | | - ... packages ...
 | | - package.json
 | - mycsharpfunction
 | | - function.json
 | | - run.csx
```

Файл host.json содержит некоторые конфигурации для конкретной среды выполнения и располагается в корневой папке приложения функции. Сведения о доступных параметрах см. в разделе [host.json ссылка](../articles/azure-functions/functions-host-json.md).

Каждая функция имеет папку, содержащую один или несколько файлов кода, конфигурации function.json и другие зависимости.
