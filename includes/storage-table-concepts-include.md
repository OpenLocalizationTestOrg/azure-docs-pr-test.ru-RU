## <a name="what-is-table-storage"></a>Что такое хранилище таблиц
В хранилище таблиц Azure содержатся большие объемы структурированных данных. Служба Hello — в хранилище данных NoSQL, которая принимает вызовы из внутри и вне hello облако Azure с проверкой подлинности. Таблицы Azure идеально подходят для хранения нереляционных структурированных данных. Самые распространенные способы использования хранилища таблиц:

* Хранение терабайтов структурированных данных с возможностью обслуживания приложений с веб-масштабированием.
* Хранение наборов данных, которые не требуют сложных соединений, внешних ключей или хранимых процедур и могут быть денормализованы для обеспечения быстрого доступа.
* Быстрый запрос данных с помощью кластерного индекса.
* Доступ к данным, используя протокол OData hello и запросы LINQ с библиотеками .NET службы данных WCF

Можно использовать таблицы хранилища toostore и запрос огромный набор нереляционных структурированных данных и таблиц масштабируется при увеличении спроса.
