Хранилище озера данных Azure — это крупномасштабный репозиторий корпоративного уровня для рабочих нагрузок анализа больших данных. Хранилище Озера данных позволяет toocapture данных любого размера, типа и приема скорости в одном месте для ведения операционной и исследовательской аналитики. Нет не toohello ограничить объем данных, которые можно хранить в хранилище Озера данных учетной записи.

| **Ресурс** | **Ограничение по умолчанию** | **Комментарии** |
| --- | --- | --- |
| Максимальное количество учетных записей Data Lake Store на подписку в регионе. |10 | Обратитесь в службу поддержки toorequest увеличение предела |
| Максимальное число списков ACL для доступа на файл или папку |32 | Это строгое ограничение. Использование групп toomanage доступа с меньшим числом записей |
| Максимальное число списков ACL по умолчанию для доступа на файл или папку |32 | Это строгое ограничение. Использование групп toomanage доступа с меньшим числом записей |
