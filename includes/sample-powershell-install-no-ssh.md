В этом примере требуется Azure PowerShell версии модуля 5.1.1 или более поздней версии. Чтобы узнать версию, выполните команду ` Get-Module -ListAvailable AzureRM`. Если вам необходимо выполнить установку или обновление, см. статью [об установке модуля Azure PowerShell](/powershell/azure/install-azurerm-ps). 

Выполните команду `Login-AzureRmAccount`, чтобы создать подключение к Azure.
