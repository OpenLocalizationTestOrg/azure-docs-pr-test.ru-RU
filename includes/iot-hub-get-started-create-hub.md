## <a name="create-an-iot-hub"></a>Создание центра IoT
Создайте центр Интернета вещей, к которому будет подключено приложение, имитирующее устройство. Ниже показано, как это сделать с помощью портала Azure.

1. Войдите на [портал Azure][lnk-portal].
1. На навигационной панели щелкните **Создать** > **Интернет вещей** > **Центр Интернета вещей**.
   
    ![Навигационная панель портала Azure][1]
1. В колонке **Центр IoT** выберите конфигурацию для центра IoT.
   
    ![Колонка "Центр IoT"][2]
   
   1. В поле **Имя** введите имя центра IoT. После проверки **имени** рядом с полем **Имя** появится зеленая галочка.
    [!INCLUDE [iot-hub-pii-note-naming-hub](iot-hub-pii-note-naming-hub.md)]
   
   1. Выберите [уровень цен и масштабирования][lnk-pricing]. Для этого руководства задавать определенный уровень не нужно. В этом руководстве используется уровень F1 (бесплатный).
   1. В разделе **Группа ресурсов** создайте новую группу ресурсов или выберите существующую. Дополнительные сведения см. в статье [Управление ресурсами Azure через портал][lnk-resource-groups].
   1. **Расположение**— выберите расположение для размещения центра IoT. В этом руководстве следует выбрать ближайшее к вам расположение.
1. Выбрав параметры конфигурации центра IoT, нажмите кнопку **Создать**.  Создание центра IoT может занять несколько минут. Чтобы проверить состояние, следите за ходом выполнения на начальной панели или панели уведомлений.
   
    ![Состояние нового центра IoT][3]
1. Когда Центр Интернета вещей будет создан, щелкните на портале Azure новую плитку для центра, чтобы открыть соответствующую колонку. Запишите **имя узла**, а затем щелкните **Политики общего доступа**.
   
    ![Колонка нового центра IoT][4]
1. В колонке **Политики общего доступа** выберите политику **iothubowner**, а затем скопируйте и запишите строку подключения в колонке **iothubowner**. Дополнительные сведения см. в разделе [Контроль доступа][lnk-access-control] в руководстве для разработчиков Центра Интернета вещей Azure.
   
    ![Колонка "Политики общего доступа"][5]
