### <a name="prerequisites"></a>Технические условия
* Учетная запись [Box](http://box.com).  

Перед использованием учетной записи Box в приложении логики необходимо авторизовать приложение логики для подключения к учетной записи Box. Это легко сделать в приложении логики на портале Azure.  

Ниже приведены инструкции по авторизации приложения логики для подключения к учетной записи Box.  

1. Чтобы создать подключение к Box, в конструкторе приложений логики в раскрывающемся списке выберите параметр **Показать API, управляемые Майкрософт**, а затем введите в поле поиска *Box*. Выберите триггер или действие, которые хотите использовать:   
   ![Этап создания подключения Box](./media/connectors-create-api-box/box-1.png)  
2. Если вы ранее не создавали подключения к Box, вам будет предложено ввести учетные данные Box. Эти учетные данные будут использоваться для авторизации приложения логики, чтобы оно могло подключиться и получить доступ к данным вашей учетной записи Box.  
   ![Этап создания подключения Box](./media/connectors-create-api-box/box-2.png)  
3. Укажите имя пользователя и пароль своей учетной записи Box, чтобы авторизовать приложение логики.  
   ![Этап создания подключения Box](./media/connectors-create-api-box/box-3.png)  
4. Разрешите подключение к Box.  
   ![Этап создания подключения Box](./media/connectors-create-api-box/box-4.png)  
5. Подключение создано, и теперь вы можете перейти к другим действиям в приложении логики:   
   ![Этап создания подключения Box](./media/connectors-create-api-box/box-5.png)
