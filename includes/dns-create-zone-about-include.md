Зоны DNS-записей DNS используется toohost hello для определенного домена. toostart размещение домена в Azure DNS необходимо toocreate зону DNS для этого имени домена. Каждая запись DNS для вашего домена создается внутри этой зоны DNS.

Например hello доменом «contoso.com» может содержать несколько записей DNS, такие как «mail.contoso.com» (для почтового сервера) и «www.contoso.com» (для веб-сайта).

При создании зоны DNS в Azure DNS учитывайте следующее.

* Имя Hello hello зону должно быть уникальным в пределах группы ресурсов hello и hello зоны не должен уже существовать. В противном случае происходит сбой операции hello.
* Здравствуйте, именем зоны можно использовать повторно в другой группе ресурсов или другой подписке Azure.
* Когда используют несколько зон hello таким же именем, каждому экземпляру присваивается другое имя адреса сервера. Только один набор адресов можно настроить с помощью регистратора доменных имен hello.

> [!NOTE]
> У вас tooown toocreate имя домена DNS-зоны с таким именем домена в Azure DNS. Однако следует соблюдать tooown hello DNS tooconfigure hello Azure DNS-серверов как hello правильное имя серверы hello доменного имени с помощью регистратора доменных имен hello.
