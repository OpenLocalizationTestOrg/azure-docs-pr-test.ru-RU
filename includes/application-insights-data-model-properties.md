Коллекция пар "имя — значение" для пользовательских свойств. Эта коллекция является стандартной телеметрии используется tooextend с hello пользовательские измерения. Примеры включают слот развертывания, связанный с данными телеметрии или ее объектом, например порядковым номером. 

Максимальная длина ключа: 150 Максимальная длина значения: 8192
