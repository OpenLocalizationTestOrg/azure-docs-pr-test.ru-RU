## <a name="error-response-format"></a>Формат сообщения об ошибке 
Виртуальные машины Azure используйте hello следующий формат JSON для ответа об ошибке:

```json
{
  "status": "status code",
  "error": {
    "code":"Top level error code",
    "message":"Top level error message",
    "details":[
     {
      "code":"Inner evel error code",
      "message":"Inner level error message"
     }
    ]
   }
}
```

Сообщение об ошибке всегда содержит код состояния и объект ошибки. Каждый объект ошибки всегда содержит код ошибки и сообщение об ошибке. Если hello виртуальная машина создается с помощью шаблона, ошибка hello объекта также содержит раздел сведений, содержащий внутреннего уровня кодов ошибок и сообщений. Как правило hello большинство внутренний уровень сообщений об ошибках — hello корневой сбоя.
