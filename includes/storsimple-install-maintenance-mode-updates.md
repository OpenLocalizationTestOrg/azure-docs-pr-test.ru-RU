#### <a name="tooinstall-maintenance-mode-updates-via-windows-powershell-for-storsimple"></a>обновления в режиме обслуживания tooinstall через Windows PowerShell для StorSimple
1. Если вы еще не сделали этого, доступ к последовательной консоли устройства hello и выберите параметр 1 **войти с полным доступом**. 
2. Введите пароль hello. пароль по умолчанию Hello — **Password1**.
3. Hello командной строки введите следующую команду:
   
     `Get-HcsUpdateAvailability` 
4. При наличии обновлений и обновлений hello, нарушающие работу или нет ли вы получите уведомление. tooapply обновления нарушают работу, необходимо tooput hello устройства в режим обслуживания. Инструкции см. в разделе [Шаг 2. Вход в режим обслуживания](../articles/storsimple/storsimple-update-device.md#step2).
5. Если ваше устройство находится в режиме обслуживания hello командной строки, введите:`Start-HcsUpdate`
6. После этого введите подтверждение для применения этих исправлений. После подтверждения обновления hello, они будут установлены на контроллер hello, выполняется доступ. После установки обновления hello hello контроллер перезагрузится. 
7. Мониторинг состояния hello обновлений. Тип:
   
    `Get-HcsUpdateStatus`
   
    Если hello `RunInProgress` — `True`, обновление hello она все еще выполняется. Если `RunInProgress` — `False`, он указывает, что hello Обновление завершено.  
8. После обновления hello установлен на текущем контроллере hello и его перезапуска, подключитесь toohello другому контроллеру и выполните шаги с 1 по 6.
9. После обновления обоих контроллеров выйдите из режима обслуживания. Инструкции см. в разделе [Шаг 4. Выход из режима обслуживания](../articles/storsimple/storsimple-update-device.md#step4).
