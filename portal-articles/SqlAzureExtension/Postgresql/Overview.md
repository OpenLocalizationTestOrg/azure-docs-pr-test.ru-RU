# <a name="azure-database-for-postgresql-documentation"></a>Документация по базе данных Azure для PostgreSQL

База данных Azure для PostgreSQL — это служба реляционной базы данных на основе СУБД PostgreSQL с открытым исходным кодом. Эта полностью управляемая база данных как услуга может обслуживать критически важные рабочие нагрузки с прогнозируемой производительностью и безопасностью, а также высоким уровнем доступности и динамическим масштабированием.  Разрабатывайте приложения на основе базы данных Azure для PostgreSQL, используя привычные средства с открытым исходным кодом и платформу.  Узнайте, как использовать базу данных Azure для PostgreSQL, ознакомившись с нашими шаблонами быстрого запуска, примерами и руководствами.
