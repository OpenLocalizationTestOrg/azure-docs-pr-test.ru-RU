# <a name="continuous-deployment-for-azure-functions"></a>Непрерывное развертывание для Функций Azure
Функции Azure позволяет легко toodeploy функции приложения с помощью непрерывной интеграции служб приложений. Компонент функции интегрируется с BitBucket, Dropbox, GitHub и Visual Studio Team Services (VSTS). Это обеспечивает рабочий процесс, где кода функции обновления, выполненные с помощью одного из этих tooAzure интегрированные службы триггера развертывания. Если новые функции tooAzure, начните с [Общие сведения о функциях Azure](functions-overview.md).

Непрерывное развертывание очень удобно для проектов, которые часто обновляются несколькими участниками. Этот процесс также обеспечивает поддержку системы управления версиями в коде функций. в настоящее время поддерживаются следующие источники развертывания Hello:

* [Bitbucket;](https://bitbucket.org/)

* Внешний репозиторий (Git или Mercurial)

* [GitHub](https://github.com)
* [OneDrive](https://onedrive.live.com/)
* [Visual Studio Team Services](https://www.visualstudio.com/team-services/)

Развертывания настраиваются на основе приложения-функции. После включения непрерывного развертывания toofunction код доступа на портале hello задано слишком*только для чтения*.
