## <a name="how-do-the-azure-stack-usage-apis-compare-to-the-azure-usage-apihttpsmsdnmicrosoftcomlibraryazure1ea5b323-54bb-423d-916f-190de96c6a3c-currently-in-public-preview"></a>Как API-интерфейсов Azure стека использования сравнения для [API использования Azure](https://msdn.microsoft.com/library/azure/1ea5b323-54bb-423d-916f-190de96c6a3c) (в настоящее время в общедоступной предварительной версии)?
* Использование API клиента согласуется с API Azure, за одним исключением: *showDetails* флаг в настоящее время не поддерживается в Azure стека.
* API использования поставщика применяется только к Azure стека.
* В настоящее время [RateCard API](https://msdn.microsoft.com/en-us/library/azure/mt219004.aspx) , доступные в Azure недоступна в Azure стека.

## <a name="what-is-the-difference-between-usage-time-and-reported-time"></a>Какова разница между временем использования и сообщил о времени?
Отчеты об использовании данных имеет два основных времени значения:

* **Отчет времени**. Время, использование событий ввода использования системы
* **Время использования**. Если ресурс Azure стек был затраченное время

Может появиться расхождение в значения для использования и сообщил о времени для конкретного события. Задержка может длиться в течение нескольких часов в любой среде.

В настоящее время вы можете запросить *только по времени сообщила*.
