# <a name="subscribe-to-an-offer"></a>Подписка на предложение
После запуска [создать предложение](azure-stack-create-offer.md), проверить, что клиенты могут создать подписку.

1. [Войдите в](azure-stack-connect-azure-stack.md) стек Azure клиента портала (https://portal.local.azurestack.external) и нажмите кнопку **получить подписку**.

   ![](media/azure-stack-subscribe-plan-provision-vm/image01.png)
2. В **отображаемое имя** поле введите имя для вашей подписки, щелкните **предлагают**, выберите один из предложения в **выберите предложение** и, при необходимости нажмите кнопку  **Создание**.

   ![](media/azure-stack-subscribe-plan-provision-vm/image02.png)
3. Чтобы просмотреть созданную подписку, щелкните **дополнительные службы**, нажмите кнопку **подписки**, затем нажмите кнопку в новую подписку.  

После оформления подписки на предложение, обновите портал, чтобы узнать, какие службы являются частью новой подписки.

## <a name="subscribe-to-an-add-on-plan"></a>Подписаться на план надстройку
Если предложение план надстройку, клиентам можно добавить их в свою подписку в любое время.  

1. На портале клиента, выберите **дополнительные службы** > **подписки**.

2. Щелкните подписку > **добавить план** и выберите план надстройку.
