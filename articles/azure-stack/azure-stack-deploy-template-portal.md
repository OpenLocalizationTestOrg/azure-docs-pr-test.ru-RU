# <a name="deploy-templates-using-the-azure-stack-portal"></a>Развертывание шаблонов с помощью портала Azure Stack
Портал можно используйте для развертывания пакета средств разработки Azure стека шаблонов диспетчера ресурсов Azure.

Шаблоны диспетчера ресурсов, развертывание и предоставление все ресурсы для приложения в один, согласованной операции.

1. Войдите портал, нажмите кнопку **New**, нажмите кнопку **настраиваемый**и нажмите кнопку **развертывания шаблона**.
2. Нажмите кнопку **изменить шаблон**, вставьте код шаблона JSON в колонке и нажмите кнопку **Сохранить**.
3. Нажмите кнопку **изменить параметры**, введите значения для параметров в списке и нажмите кнопку **ОК**.
4. Нажмите кнопку **подписки**, выберите подписку, которую нужно использовать, а затем нажмите кнопку **ОК**.
5. Нажмите кнопку **группы ресурсов**, выберите существующую группу ресурсов или создайте новый и нажмите кнопку **ОК**.
6. Щелкните **Создать**. Новый элемент на панели мониторинга отслеживает ход выполнения развертывания вашего шаблона.
