# <a name="deploy-templates-using-hello-azure-stack-portal"></a>Развертывание шаблонов с помощью портала Azure стека hello
Используйте hello портала toodeploy диспетчера ресурсов Azure шаблоны toohello стека Azure пакет средств разработки.

Шаблоны диспетчера ресурсов, развертывание и предоставление всех hello ресурсы для приложения в один, согласованной операции.

1. Войдите в портал toohello, нажмите кнопку **New**, нажмите кнопку **настраиваемый**и нажмите кнопку **развертывания шаблона**.
2. Нажмите кнопку **изменить шаблон**, вставьте код шаблона JSON в колонке hello и нажмите кнопку **Сохранить**.
3. Нажмите кнопку **изменить параметры**, введите значения для параметров hello в списке и нажмите кнопку **ОК**.
4. Нажмите кнопку **подписки**, выберите подписку hello toouse и нажмите кнопку **ОК**.
5. Нажмите кнопку **группы ресурсов**, выберите существующую группу ресурсов или создайте новый и нажмите кнопку **ОК**.
6. Щелкните **Создать**. Новую плитку на панели мониторинга hello отслеживает ход выполнения hello развертывания шаблона.
