# <a name="add-users-in-the-azure-stack-development-kit"></a>Добавление пользователей в пакете средств разработки стек Azure

Чтобы добавить дополнительных пользователей для развертывания пакета средств разработки, необходимо добавить их к каталогу пакета средств разработки Azure стека с помощью консоли управления на главном компьютере стек Azure.
1.  На главном компьютере стека Azure откройте консоль управления.
2.  Нажмите кнопку **файл > Добавить или удалить оснастку**.
3.  Выберите **Active Directory-пользователи и компьютеры** > **AzureStack.local** > **пользователей**.
4.  Нажмите кнопку **действия** > **новый** > **пользователя**.
5.  В новый объект — пользователь окна, укажите и подтвердите пароль
6.  Нажмите кнопку **Далее** finalize значения и нажмите кнопку Готово, чтобы создать пользователя.
