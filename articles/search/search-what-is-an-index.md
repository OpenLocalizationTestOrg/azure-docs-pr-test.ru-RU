## <a name="what-is-an-index"></a>Что такое индекс
*Индекс* — это постоянное хранилище *документов* и других конструкций, используемое службой поиска Azure. Документ — это единый блок доступных для поиска данных в индексе. Например, в интернет-магазине для каждой позиции продажи есть определенный документ, в агентстве новостей есть такой документ для каждой статьи и т. д.
