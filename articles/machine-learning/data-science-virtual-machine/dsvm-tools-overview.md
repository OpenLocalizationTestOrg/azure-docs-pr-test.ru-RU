# <a name="get-to-know-your-data-science-virtual-machine"></a>Общие сведения о виртуальной машине для обработки и анализа данных

Виртуальная машина для обработки и анализа данных (**DSVM**) содержит большой ряд популярных средств обработки и анализа данных, средств машинного обучения и AI, а также платформ данных. Это было настроено в конфигурациях, которые тщательно протестированы специалистами по обработке и анализу данных и разработчиками в корпорации Майкрософт, а также широким сообществом по обработке и анализу данных, чтобы обеспечить стабильность и общую устойчивость.

DSVM также содержит несколько примеров кода и пошаговые руководства в виде записных книжек Jupyter, которые можно выполнить локально. Имеется также документация для облегчения понимания этих образцов. 

Средства, платформы, служебные программы и примеры, включенные в DSVM, разделены на следующие разделы:

1. **[Платформы глубокого обучения и искусственного интеллекта](dsvm-deep-learning-ai-frameworks.md)**
2. **[Средства машинного обучения, обработки и анализа данных](dsvm-ml-data-science-tools.md)**
3. **[Языки, поддерживаемые на виртуальных машинах для обработки и анализа данных](dsvm-languages.md)**
4. **[Средства разработки на виртуальных машинах для обработки данных](dsvm-tools-development.md)**
5. **[Средства приема данных виртуальных машин для обработки и анализа данных](dsvm-tools-ingestion.md)**
6. **[Средства исследования и визуализации данных на виртуальной машине для обработки и анализа данных](dsvm-tools-explore-and-visualize.md)**
7. **[Платформы данных](dsvm-data-platforms.md)**
8. **[Примеры для виртуальных машин для обработки и анализа данных](dsvm-samples-and-walkthroughs.md)**
