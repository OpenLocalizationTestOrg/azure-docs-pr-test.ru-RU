# <a name="customize-an-assessment"></a>Настройка оценки

Служба [Миграция Azure](migrate-overview.md) создает оценки с параметрами по умолчанию. После создания оценки можно изменить ее параметры по умолчанию, выполнив инструкции в этой статье.


## <a name="edit-assessment-values"></a>Изменение значений оценки

1. В проекте службы "Миграция Azure" на странице **Оценки** выберите оценку и щелкните **Изменить свойства**.
2. Измените параметры в соответствии со следующей таблицей.

    **Параметр** | **Дополнительные сведения** | **По умолчанию**
    --- | --- | ---
    **Целевое расположение** | Расположение Azure для миграции. |  "Западная часть США 2" — это расположение по умолчанию.
    **Избыточность хранилища** | Тип хранилища для виртуальных машин Azure после миграции. | Сейчас поддерживается только репликация [локально избыточного хранилища (LRS)](../storage/common/storage-redundancy.md#locally-redundant-storage).
    **Фактор комфорта** | Фактор комфорта — это буфер, который используется во время оценки. Используйте его, чтобы учесть такие факторы, как сезонное использование, небольшой журнал производительности и вероятное увеличение использования в будущем. | Значение по умолчанию — "1,3x".
    **Журнал производительности** | Период времени, используемый при оценке журнала производительности. | По умолчанию это один месяц.
    **Использование процентиля** | Значение процентиля, учитываемое для журнала производительности. | Значение по умолчанию — 95 %.
    **Ценовая категория** | Можно указать [ценовую категорию](https://azure.microsoft.com/blog/basic-tier-virtual-machines-2/) для виртуальной машины.  | По умолчанию применяется категория [Стандартный](../virtual-machines/windows/sizes-general.md).
    **ПРЕДЛОЖЕНИЕ** | Применяемые [предложения Azure](https://azure.microsoft.com/support/legal/offer-details/). | По умолчанию используется [Оплата по мере использования](https://azure.microsoft.com/offers/ms-azr-0003p/).
    **Валюта** | Валюта для выставления счетов. | По умолчанию это доллары США.
    **Скидка (%)** | Любая относящаяся к подписке скидка, предоставляемая в рамках какого-либо предложения. | Значение по умолчанию — 0 %.
    **Преимущество гибридного использования Azure** | Указывает, зарегистрированы ли вы в программе [Преимущество гибридного использования Azure](https://azure.microsoft.com/pricing/hybrid-use-benefit/). Если установлено значение "Да", то для виртуальных машин Windows действуют цены, отличные от цен Microsoft Azure. | Значение по умолчанию — "Да".

3. Нажмите кнопку **Сохранить**, чтобы обновить оценку.
