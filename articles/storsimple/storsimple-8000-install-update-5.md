# <a name="install-update-5-on-your-storsimple-device"></a>Установка обновления 5 на устройство StorSimple

## <a name="overview"></a>Обзор

В этом руководстве объясняется, как с помощью портала Azure и применения исправления установить обновление 5 на устройство StorSimple с программным обеспечением более ранней версии. Метод исправления используется, если шлюз настроен для сетевого интерфейса устройства StorSimple, отличного от DATA 0, и вы пытаетесь провести обновление с версии ПО до обновления 1.

Обновление 5 включает в себя обновления программного обеспечения для устройств, Storport и Spaceport, обновления безопасности операционной системы и другие обновления операционной системы, а также обновления встроенного ПО диска.  Обновления программного обеспечения для устройств, Spaceport, Storport, обновления безопасности и другие обновления операционной системы не нарушают работу устройства. Обычные обновления или обновления, которые не влияют на работу устройства, можно установить с помощью портала Azure или применив исправление. Обновления встроенного ПО диска нарушают работу устройства и могут применяться, когда устройство находится в режиме обслуживания. Для этого используется метод с установкой исправления через интерфейс Windows PowerShell на устройстве.

> [!IMPORTANT]
> * Набор предварительных проверок (выполняются вручную и автоматически перед установкой), которые позволяют определить работоспособность устройства, то есть состояние его оборудования и качество сетевого подключения. Эти проверки выполняются, только если обновления устанавливаются через портал Azure.
> * Мы рекомендуем устанавливать обновления программного обеспечения и другие обычные обновления с помощью портала Azure. Устанавливайте обновления через интерфейс Windows PowerShell на устройстве только в том случае, если предварительная проверка шлюза возвращает ошибку на портале. В зависимости от версии обновляемого ПО установка обновлений может занять от 4 часов. Установка обновлений в режиме обслуживания должна выполняться только через интерфейс Windows PowerShell на устройстве. Так как обновления в режиме обслуживания нарушают работу устройства, устройство на некоторое время становится недоступным.
> * Если дополнительно выполняется StorSimple Snapshot Manager, то перед обновлением устройства убедитесь, что проведено обновление версии Snapshot Manager до обновления 5.

## <a name="install-update-5-via-the-azure-portal"></a>Установка обновления 5 с помощью портала Azure
Выполните следующие действия, чтобы установить [обновление 5](storsimple-update5-release-notes.md) на устройство.

> [!NOTE]
> Майкрософт получает дополнительную диагностическую информацию от устройства. Поэтому при обнаружении нашей операционной группой проблем на устройствах мы имеем более широкие возможности для сбора информации с устройства и диагностики проблем.

[!INCLUDE [storsimple-8000-install-update4-via-portal](../../includes/storsimple-8000-install-update5-via-portal.md)]

Убедитесь, что на устройстве установлено **обновление 5 для StorSimple серии 8000 (6.3.9600.17845)**. Также должна измениться **дата последнего обновления**.

* Теперь вы увидите, что доступны обновления режима обслуживания (это сообщение может отображаться до 24 часов после установки обновлений). Обновления режима обслуживания являются критическими — они прерывают рабочие процессы устройства и могут быть реализованы только через интерфейс Windows PowerShell устройства.

* Скачайте обновления режима обслуживания, выполнив действия, описанные в подразделе [Загрузка исправления](#to-download-hotfixes), чтобы найти и скачать исправление KB4011837, которое устанавливает обновления встроенного ПО диска (другие обновления уже должны быть установлены). Выполните действия, описанные в разделе [Установка и проверка исправления режима обслуживания](#to-install-and-verify-maintenance-mode-hotfixes) для установки обновлений режима обслуживания.

## <a name="install-update-5-as-a-hotfix"></a>Установка обновления 5 в виде исправления


Этот метод подходит для обновления таких версий программного обеспечения:

* обновление 0.1, 0.2, 0.3;
* обновление 1, 1.1, 1.2;
* обновление 2, 2.1, 2.2.
* обновление 3, 3.1.
* Обновление 4

> [!NOTE] 
> Мы рекомендуем установить обновление 5 с помощью портала Azure. Эта процедура применима, когда при попытке установить обновления через портал Azure проверка шлюза возвращает ошибку. Проверка завершается ошибкой, когда шлюз назначен сетевому интерфейсу, отличному от DATA 0, а на устройстве используется более ранняя версия программного обеспечения (до обновления 1).

Метод исправления состоит из следующих трех шагов:

1. Загрузите исправления из каталога Центра обновления Майкрософт.
2. Установите и проверьте исправления обычного режима.
3. Установите и проверьте исправления режима обслуживания.
