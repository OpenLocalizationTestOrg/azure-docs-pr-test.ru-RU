# <a name="install-update-22-on-your-storsimple-device"></a>Установка обновления 2.2 на устройство StorSimple
## <a name="overview"></a>Обзор
В этом учебнике описывается установка обновления 2.2 на устройстве StorSimple, на котором используется программное обеспечение более ранней версии, посредством классического портала Azure и метода с установкой исправления. Метод исправления используется, если шлюз настроен для сетевого интерфейса устройства StorSimple, отличного от DATA 0, и вы пытаетесь провести обновление с версии ПО до обновления 1.

В обновление 2.2 входят обновления ПО устройства, WMI и iSCSI. При обновлении версии 2.1 применять нужно только обновление программного обеспечения устройства. В случае обновления с версии до обновления 2 также понадобится применить обновления драйвера LSI, Spaceport, Storport и встроенного ПО диска. Обновления программного обеспечения устройства, WMI, iSCSI, Spaceport, Storport и драйвера LSI не нарушают работу устройства и могут устанавливаться через классический портал Azure. Обновления встроенного ПО диска нарушают работу устройства и могут устанавливаться только через интерфейс Windows PowerShell на устройстве. 

> [!IMPORTANT]
> * Набор предварительных проверок (выполняются вручную и автоматически перед установкой), которые позволяют определить работоспособность устройства, то есть состояние его оборудования и качество сетевого подключения. Эти проверки выполняются только в том случае, если обновления устанавливаются через классический портал Azure.
> * Рекомендуем устанавливать обновления программного обеспечения и драйвера с помощью классического портала Azure. Устанавливайте обновления через интерфейс Windows PowerShell на устройстве только в том случае, если предварительная проверка шлюза возвращает ошибку на портале. В зависимости от версии обновляемого ПО установка обновлений может занять 1,5–2,5 часа. Установка обновлений в режиме обслуживания должна выполняться только через интерфейс Windows PowerShell на устройстве. Так как обновления в режиме обслуживания нарушают работу устройства, устройство некоторое время будет недоступно.
> * Если выполняется дополнительный диспетчер моментальных снимков StorSimple, то перед обновлением устройства убедитесь, что установлено обновление 2.2 диспетчера моментальных снимков.
> 
>

## <a name="install-update-22-via-the-azure-classic-portal"></a>Установка обновления 2.2 с помощью классического портала Azure
Выполните следующие действия, чтобы установить [обновление 2.2](storsimple-update21-release-notes.md)на устройство.

> [!NOTE]
> В случае применения обновления 2 или более поздней версии (включая обновление 2.1) корпорация Майкрософт сможет получить дополнительные диагностические сведения с устройства. Поэтому при обнаружении нашей операционной группой проблем на устройствах мы имеем более широкие возможности для сбора информации с устройства и диагностики проблем. Соглашаясь на установку обновления 2 или более поздней версии, вы разрешаете нам предоставить эту профилактическую поддержку.
> 
> 

[!INCLUDE [storsimple-install-update2-via-portal](../../includes/storsimple-install-update2-via-portal.md)]

1. Убедитесь, что на устройстве установлено **обновление 2.2 для StorSimple серии 8000 (6.3.9600.17708)**. Также должна измениться **Дата последнего обновления** . 
   
   При обновлении с версии до обновления 2 вы также увидите, что доступны обновления режима обслуживания (это сообщение может отображаться до 24 часов после установки обновлений).
   
   Обновления режима обслуживания являются критическими — они прерывают рабочие процессы устройства и могут быть реализованы только через интерфейс Windows PowerShell устройства. В некоторых случаях при выполнении обновления 1.2 встроенное ПО диска уже может быть последней версии. Тогда установка каких-либо обновлений режима обслуживания не требуется.
   
   Если обновление 2 установлено, устройство должно быть в обновленном состоянии. Оставшиеся действия можно пропустить.
2. Скачайте обновления режима обслуживания, выполнив действия, описанные в разделе [Загрузка исправления](#to-download-hotfixes) , чтобы найти и скачать исправление KB3121899, которое устанавливает обновления встроенного ПО диска (другие обновления уже должны быть установлены).
3. Выполните действия, описанные в разделе [Установка и проверка исправления режима обслуживания](#to-install-and-verify-maintenance-mode-hotfixes) для установки обновлений режима обслуживания. 

## <a name="install-update-22-as-a-hotfix"></a>Установка обновления 2.2 в виде исправления
Эту процедуру следует использовать в том случае, если при попытке установить обновления через классический портал Azure проверка шлюза возвращает ошибку. Проверка завершается ошибкой, так как ваш шлюз назначен сетевому интерфейсу, отличному от DATA 0, а на устройстве используется более ранняя версия программного обеспечения (до обновления 1).

Этот метод подходит для обновления таких версий программного обеспечения:

* обновление 0.1, 0.2, 0.3;
* обновление 1, 1.1, 1.2;
* обновление 2, 2.1. 

> [!IMPORTANT]
> * Если на вашем устройстве используется выпуск GA, обратитесь в [службу поддержки Майкрософт](storsimple-contact-microsoft-support.md) за помощью в установке обновления.
> 
> 

Метод исправления состоит из следующих трех шагов:

* Загрузите исправления из каталога Центра обновления Майкрософт.
* Установите и проверьте исправления обычного режима.
* Установите и проверьте исправление режима обслуживания (только при обновлении с версии программного обеспечения до обновления 2).

#### <a name="download-updates-for-a-device-running-update-21-software"></a>Скачивание обновлений для устройства под управлением ПО с обновлением 2.1
**Если на устройстве установлено обновление 2.1**, то необходимо скачать только обновление ПО устройства KB3179904. Установите только двоичный файл, имя которого начинается с all-hcsmdssoftwareudpate. Не устанавливайте обновление агента Cis и MDS, имя которого начинается с `all-cismdsagentupdatebundle`. В противном случае произойдет сбой. Это обновление не нарушит работу, операции ввода-вывода не будут прерваны, и устройство не будет простаивать.

#### <a name="download-updates-for-a-device-running-update-2-software"></a>Загрузка обновлений для устройства под управлением программного обеспечения обновления 2
**Если устройство работает под управлением ПО с обновлением 2**, то необходимо скачать и установить следующие исправления в указанном порядке.

| Порядок | КБ | Описание | Тип обновления | Время установки |
| --- | --- | --- | --- | --- |
| 1. |KB3179904 |Обновление программного обеспечения &#42; |Регулярное,  <br></br>не нарушает работу |~ 45 мин |
| 2. |KB3146621 |Пакет iSCSI |Регулярное,  <br></br>не нарушает работу |~ 20 мин |
| 3. |KB3103616 |Пакет WMI |Регулярное,  <br></br>не нарушает работу |~ 12 мин |

 &#42; *Обратите внимание, что обновление программного обеспечения состоит из двух двоичных файлов: обновления программного обеспечения устройства, имя которого начинается с `all-hcsmdssoftwareupdate`, и обновления агента Cis и MDS, имя которого начинается с `all-cismdsagentupdatebundle`. Обновление программного обеспечения устройства необходимо установить раньше агента Cis и MDS. Кроме того, необходимо перезапустить активный контроллер с помощью командлета `Restart-HcsController` после применения обновления агента Cis и MDS (и перед применением оставшихся обновлений).* 

#### <a name="download-updates-for-a-device-running-pre-update-2-software"></a>Загрузка обновлений для устройства под управлением ПО до обновления 2
**Если устройство работает под управлением версии 0.2, 0.3, 1.0 или 1.1**, то кроме обновлений программного обеспечения, iSCSI и WMI, необходимо скачать и установить обновление драйвера LSI и встроенного ПО. Это обновление уже установлено, если ваше устройство работает под управлением обновления версии 1.2 или 2. 

| Порядок | КБ | Описание | Тип обновления | Время установки |
| --- | --- | --- | --- | --- |
| 4. |KB3121900 |Драйвер LSI и встроенное ПО |Регулярное,  <br></br>не нарушает работу |~ 20 мин |

<br></br>
**Если устройство работает под управлением версии 0.2, 0.3, 1.0, 1.1 или 1.2**, то необходимо скачать и установить исправление для Spaceport и Storport. Эти обновления уже установлены, если ваше устройство работает под управлением обновления версии 2.

| Порядок | КБ | Описание | Тип обновления | Время установки |
| --- | --- | --- | --- | --- |
| 5. |KB3090322 |Исправление Spaceport  </br> Windows Server 2012 R2 |Регулярное,  <br></br>не нарушает работу |~ 20 мин |
| 6. |KB3080728 |Исправление Storport  </br> Windows Server 2012 R2 |Регулярное,  <br></br>не нарушает работу |~ 20 мин |

<br></br>
Вам также понадобится установить обновления встроенного ПО диска. Вы можете проверить, необходимо ли обновление встроенного ПО диска, выполнив командлет `Get-HcsFirmwareVersion` . Если вы используете встроенное ПО версии `XMGG`, `XGEG`, `KZ50`, `F6C2` или `VR08`, то эти обновления устанавливать не нужно.

| Порядок | КБ | Описание | Тип обновления | Время установки |
| --- | --- | --- | --- | --- |
| 7. |KB3121899 |Встроенное ПО диска |Обслуживание,  <br></br>нарушает работу |~ 30 мин |

<br></br>

> [!IMPORTANT]
> * Эту процедуру требуется выполнить только один раз для установки обновления 2.2. Вы можете использовать классический портал Azure для установки всех последующих обновлений.
> * Если обновление 2 установлено, общее время установки при обновлении составляет примерно 1,5 часа.
> * Прежде чем использовать эту процедуру для установки обновления, убедитесь, что оба контроллера устройства подключены к сети и все аппаратные компоненты находятся в работоспособном состоянии.
> 
>
