# <a name="storsimple-8000-series-update-22-release-notes"></a>Заметки о выпуске, посвященные обновлению 2.2 для устройств серии StorSimple 8000.
> [!NOTE]
> Классический портал StorSimple устарел. Диспетчеры устройств StorSimple автоматически перейдут на новый портал Azure в соответствии с графиком устаревания. Вы получите сообщение электронной почты и уведомление с портала, касающиеся этого перехода. Этот документ скоро также перестанет использоваться. Сведения, связанные с переходом, см. в [ответах на вопросы о перемещении на портал Azure](storsimple-8000-move-azure-portal-faq.md).

## <a name="overview"></a>Обзор
В приведенных ниже заметках о выпуске описаны новые функции и критические открытые проблемы в обновлении 2.2 для устройств серии StorSimple 8000. Здесь также содержится список обновлений программного обеспечения StorSimple для этого выпуска.

Обновление 2.2 можно применить к любому устройству StorSimple под управлением ПО общедоступной версии или версии с обновлениями версий 0.1–2.1. Версия устройства, связанная с обновлением 2.2 — 6.3.9600.17708.

Перед развертыванием обновления в решении StorSimple, просмотрите сведения, содержащиеся в заметках о выпуске.

> [!IMPORTANT]
> * В обновлении 2.2 содержатся обновления только программного обеспечения. Установка этого обновления занимает примерно 1,5–2 часа. 
> * Если вы используете обновление 2.1, то рекомендуется применить обновление 2.2 как можно скорее.
> * Для новых выпусков обновления могут отображаться не сразу, поскольку развертывание обновлений выполняется поэтапно. Подождите несколько дней и затем повторите попытку поиска обновлений, так как они станут доступными в ближайшее время.
> 
> 

## <a name="whats-new-in-update-22"></a>Новые возможности в обновлении 2.2
В обновление 2.2 внесены следующие основные улучшения:

* **Оптимизация автоматической реорганизации пространства**: при удалении данных на томах с тонкой подготовкой необходимо освобождать неиспользуемые блоки ресурсов хранения. В этом выпуске улучшен процесс реорганизации пространства из облака. Благодаря этому неиспользуемое пространство становится доступным быстрее по сравнению с предыдущими версиями.
* **Улучшения производительности моментальных снимков**. В обновлении 2.2 требуется меньше времени на обработку облачных моментальных снимков в определенных сценариях, где используются большие тома с минимальным обновлением данных или без него. Это улучшение целесообразно использовать для архивных томов.
* **Усиление защиты сбора пакета поддержки**. В этом выпуске улучшен способ сбора и передачи пакета поддержки. 
* **Улучшения надежности обновления**. В этом выпуске содержатся исправления, благодаря которым повышается надежность обновления.

## <a name="controller-and-firmware-updates-in-update-22"></a>Обновления контроллера и встроенного ПО в обновлении 2.2
В этом выпуске содержатся обновления только для программного обеспечения. Однако при обновлении с версии до обновления 2 понадобится установить обновления драйвера, Storport, Spaceport и (в некоторых случаях) встроенного ПО диска на устройстве.

Дополнительные сведения о том, как установить обновления драйвера, Storport, Spaceport и встроенного ПО диска, см. в статье [Установка обновления 2.2 на устройство StorSimple](storsimple-install-update-21.md).
