# <a name="security-playbook-in-azure-security-center-preview"></a>Сборник тренировочных заданий по обеспечению безопасности в центре безопасности Azure (предварительная версия)
Этот документ поможет вам использовать сборники тренировочных заданий по обеспечению безопасности в центре безопасности Azure для реагирования на угрозы безопасности.

## <a name="what-is-security-playbook-in-security-center"></a>Что такое сборник тренировочных заданий по обеспечению безопасности в центре безопасности
Сборник тренировочных заданий по обеспечению безопасности представляет собой набор процедур, которые могут быть выполнены в центре безопасности, когда определенный сборник запускается с помощью выбранного оповещения. Сборник тренировочных заданий по обеспечению безопасности может помочь автоматизировать и настроить реагирование на конкретное оповещение безопасности, обнаруженное центром безопасности. Сборники тренировочных заданий по обеспечению безопасности в центре безопасности основаны на [Azure Logic Apps](https://docs.microsoft.com/azure/logic-apps/logic-apps-what-are-logic-apps). Это означает, что вы можете использовать шаблоны, которые содержатся в категории безопасности в шаблонах Logic Apps, вы можете изменять их в соответствии с потребностями или создавать новые сборники тренировочных заданий с помощью [рабочего процесса Azure Logic Apps](https://docs.microsoft.com/azure/logic-apps/logic-apps-create-a-logic-app) и использовать центр безопасности в качестве триггера. 

> [!NOTE]
> Сборник тренировочных заданий использует Azure Logic Apps, поэтому плата взимается. Посетите страницу цен на [Azure Logic Apps](https://azure.microsoft.com/pricing/details/logic-apps/) для получения подробных сведений о ценообразовании. 

## <a name="how-to-create-a-security-playbook-from-security-center"></a>Как создать сборник тренировочных заданий по обеспечению безопасности в центре обеспечения безопасности
Чтобы создать сборник тренировочных заданий по обеспечению безопасности в центре обеспечения безопасности, следуйте следующим требованиям:

1.  Откройте панель мониторинга **Центр безопасности**.
2.  В разделе **Automation & Orchestration** (Автоматизация и оркестрация) в левой области щелкните пункт **Playbooks (Preview)** (Сборники тренировочных заданий (предварительная версия)).

    ![приложение логики;](./media/security-center-playbooks/security-center-playbooks-fig1.png)
 
3. На странице **Security Center - Playbooks (Preview)** (Центр безопасности. Сборники тренировочных заданий (предварительная версия)) нажмите кнопку **Добавить**.

    ![Создание приложения логики](./media/security-center-playbooks/security-center-playbooks-fig2.png)

4. На странице **Create Logic app** (Создание приложения логики) введите запрашиваемые сведения для создания приложения логики и нажмите кнопку **Создать**. По завершении создания новый сборник тренировочных заданий появится в списке. В противном случае нажмите кнопку **Обновить**. Когда сборник тренировочных заданий появится, щелкните его, чтобы начать редактирование.

    ![Создание приложения логики](./media/security-center-playbooks/security-center-playbooks-fig3.png)

5. Откроется **конструктор приложений логики**. Щелкните **Пустое приложение логики**, чтобы создать новый сборник тренировочных заданий. Можно также выбрать **Безопасность** в группе категорий и использовать один из шаблонов.
    
    ![Конструктор приложений логики](./media/security-center-playbooks/security-center-playbooks-fig4.png)

6. В поле **Search all connectors and triggers** (Поиск по всем соединителям и триггерам) введите *Azure Security Center* и выберите **When a response to an Azure Security Center alert is triggered** (Когда срабатывает реагирование на оповещение центра безопасности Azure).

    ![Триггер](./media/security-center-playbooks/security-center-playbooks-fig12.png)

7. Теперь можно определить, что произойдет при активации сборника тренировочных заданий. Можно добавить действие, логическое условие, условия для оператора switch или циклы.

    ![Конструктор приложений логики](./media/security-center-playbooks/security-center-playbooks-fig5.png)
     
## <a name="how-to-run-a-security-playbook-in-security-center"></a>Как запустить сборник тренировочных заданий по обеспечению безопасности в центре безопасности

Вы можете запустить сборник тренировочных заданий по обеспечению безопасности в центре безопасности, если требуется выполнить оркестрацию, получить дополнительные сведения из других служб или исправить неполадку. Для доступа к сборникам тренировочных заданий сделайте следующее:

1.  Откройте панель мониторинга **Центр безопасности**.
2.  В левой области в разделе **Обнаружение угроз** щелкните **Security incidents & alerts** (Инциденты безопасности и оповещения).

    ![Оповещения](./media/security-center-playbooks/security-center-playbooks-fig6.png)

3.  Щелкните оповещение, которое необходимо проанализировать.
4.  В верхней части страницы этого оповещения нажмите кнопку **Run playbooks** (Запустить сборники тренировочных заданий).

    ![Запуск сборника тренировочных заданий](./media/security-center-playbooks/security-center-playbooks-fig7.png)

5. На странице сборников тренировочных заданий выберите сборник, который требуется запустить, и нажмите кнопку **Запустить**. Если вы хотите увидеть сборник перед запуском, щелкните его, и откроется конструктор.

    ![Сборники тренировочных заданий](./media/security-center-playbooks/security-center-playbooks-fig13.png)

### <a name="history"></a>Журнал

После запуска сборника тренировочных заданий вы также можете получить доступ к предыдущим запускам и шагам, которые содержат больше сведений о состоянии ранее выполненных сборников. История контекстуализуется для каждого оповещения, а это значит, что история сборников тренировочных заданий, которую вы видите на этой странице, сопоставляется с оповещением, вызвавшим этот сборник. 

![Журнал](./media/security-center-playbooks/security-center-playbooks-fig16.png)

Для получения дополнительных сведений о выполнении конкретного сборника щелкните сборник, после чего отобразится страница выполнения приложения логики со всем рабочим процессом.

![Сведения](./media/security-center-playbooks/security-center-playbooks-fig14.png)

В этом рабочем процессе вы можете увидеть время, в течение которого выполнялся рабочий процесс, а также можно развернуть каждую задачу, чтобы увидеть результат. 

### <a name="changing-an-existing-playbook"></a>Изменение существующего сборника тренировочных заданий

Можно изменить существующий сборник в центре безопасности, чтобы добавить действие или условия. Для этого на вкладке "Сборники тренировочных заданий" нужно щелкнуть имя сборника, который вы хотите изменить, после чего откроется конструктор приложений логики.

> [!NOTE]
> Дополнительные сведения о создании собственных сборников тренировочных заданий с помощью приложения логики Azure см. в статье [Создание первого рабочего процесса приложения логики для автоматизации процессов между облачными приложениями и облачными службами](https://docs.microsoft.com/azure/logic-apps/logic-apps-create-a-logic-app#add-an-action-that-responds-to-your-trigger).
