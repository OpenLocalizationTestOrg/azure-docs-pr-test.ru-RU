# <a name="how-to-use-azure-iot-hub"></a>Использование Центра Интернета вещей в Azure

Вы можете обучиться разработке для служб Центра Интернета вещей разными способами:

* прочтите статьи с подробным описанием основных понятий и возможностей Центра Интернета вещей;
* выполните задачи в одном из руководств, посвященных различным возможностям Центра Интернета вещей.

## <a name="developer-guide"></a>Руководство разработчика по центру Azure IoT (IoT — Интернет вещей)

Разработчики могут ознакомиться с подробной документацией о Центре Интернета вещей из [руководства разработчика][lnk-devguide]. Это руководство содержит:

* подробное описание всех возможностей Центра Интернета вещей и способов их использования;
* рекомендации по выбору, если доступно несколько вариантов.

## <a name="tutorials"></a>Учебники

Если вы предпочитаете изучать определенные функции Центра Интернета вещей с помощью практических упражнений, доступно несколько руководств. Некоторые руководства описывают работу с разными языками программирования. Ниже представлен список таких руководств.

- [Обработка сообщений Центра Интернета вещей, отправляемых с устройства в облако, с использованием маршрутов (.NET)][lnk-routes-tutorial]. Из этого руководства вы узнаете, как через настройки системы можно использовать правила маршрутизации Центра Интернета вещей для перенаправления сообщений, отправляемых с устройства в облако.

- [Отправка сообщений из облака на устройство с помощью Центра Интернета вещей (.NET)][lnk-c2d-tutorial]. В этом руководстве описано, как отправлять сообщения из облака на устройства с помощью Центра Интернета вещей и получать сообщения, передаваемые из облака на устройство.

- [Передача файлов из устройств в облако в Центре Интернета вещей][lnk-upload-tutorial]. В этом руководстве описано, как использовать возможности передачи файлов Центра Интернета вещей.

- [Начало работы с двойниками устройств (Node)][lnk-twin-tutorial]. В этом руководстве описываются двойники устройств, сообщаемые свойства, требуемые свойства и теги. Двойники устройств позволяют синхронизировать значения с устройствами.

- [Использование прямых методов (Node)][lnk-methods-tutorial]. В этом руководстве описано использование прямых методов. Вы добавляете обработчик для прямого метода на виртуальном устройстве и вызываете прямой метод из Центра Интернета вещей.

- [Начало работы с управлением устройствами (Node)][lnk-dm-tutorial]. В этом руководстве описано, как использовать ключевые функции управления устройствами (например, функции двойника устройства или прямого метода). Эти функции используются для удаленной перезагрузки виртуального устройства.

- [Настройка устройств с помощью требуемых свойств][lnk-properties-tutorial]. В этом руководстве представлены сведения об удаленной настройке устройства с помощью сообщаемых свойств двойника устройства.

- [Используйте управление устройствами, чтобы запустить обновление встроенного ПО устройства (Node/Node)][lnk-jobs-tutorial]. В этом руководстве описано, как использовать ключевые функции управления устройствами (например, функции двойника устройства или прямого метода). Вы узнаете, как применять эти функции для удаленного обновления встроенного ПО на устройстве.

- [Учебник. Планирование и трансляция заданий][lnk-schedule-tutorial]. В этом руководстве показано, как использовать требуемые свойства и прямые методы для взаимодействия с несколькими устройствами в запланированное время.
