# <a name="avoid-getting-charged-for-your-azure-free-account"></a>Как не платить за бесплатную учетную запись Azure

Azure бесплатной учетной записи позволяет 200 в счете в Azure для первых 30 дней и ограниченное количество свободного служб за 12 месяцев. Дополнительные сведения см. в разделе [Azure бесплатную учетную запись](https://azure.microsoft.com/free/). В зависимости от состояния кредита могут использовать кредита или выставляется за бесплатной службы и количества.

## <a name="you-still-have-active-azure-free-account-credit"></a>У вас все еще есть деньги на счете в бесплатной учетной записи Azure 
Если по-прежнему кредита Azure еще не истек, Azure использует кредита для оплаты за бесплатной службы и количества.

## <a name="your-credit-ran-out-or-has-expired"></a>Деньги на счете в Azure закончились, или их срок действия истек
Если через 30 дней деньги на счете в Azure закончились или истек срок их действия, Azure отключит вашу подписку. Чтобы продолжить использование служб Azure, необходимо обновить подписку на подписку с оплатой по мере использования. Дополнительные сведения см. в статье [Обновление с бесплатной пробной версии или подписки Azure Microsoft Imagine до подписки с оплатой по мере использования](billing-upgrade-azure-subscription.md). После обновления, ваша подписка по-прежнему имеет доступ к бесплатным служб в течение 12 месяцев. Вы только выставляется за бесплатной службы и количества.


Рассмотрим некоторые из причин, из-за которых может взиматься плата в бесплатной учетной записи Azure.


### <a name="your-usage-exceeds-the-limits-of-free-services-included-with-your-azure-free-account"></a>Объем использования превышает ограничения бесплатной службы, включаемые в Azure бесплатной учетной записи

Вы получаете ограниченное количество бесплатных служб каждый месяц в бесплатной учетной записи Azure. Количество бесплатных служб истекает в конце месяца и не переносится на следующий месяц. Например, вы получаете 5 ГБ хранилища файлов каждый месяц. Если в месяц использовалось 2 ГБ, оставшиеся 3 ГБ не переносятся на следующий месяц. Чтобы избежать выставления счетов, потребляйте ресурсы в допустимых пределах. Дополнительные сведения об ограничениях на бесплатные службы см. в статье [Вопросы и ответы о бесплатной учетной записи Azure](https://azure.microsoft.com/free/free-account-faq/). Чтобы проверить потребление бесплатных служб, ознакомьтесь со статьей [Проверка использования бесплатных служб, включаемых в бесплатную учетную запись Azure](billing-check-free-service-usage.md). 

### <a name="you-are-using-services-not-included-for-free-with-your-azure-free-account"></a>Используются не включены бесплатно с Azure бесплатной учетной записи службы

Можно получить взимается Стандартная плата по мере использования для использования служб, которые не включены бесплатно с учетной записью Azure бесплатно. Дополнительные сведения о службах, входящих в бесплатную учетную запись, см. в статье [Вопросы и ответы о бесплатной учетной записи Azure](https://azure.microsoft.com/free/free-account-faq/). Можно проверить использование службы, которые используются на портале Azure или через файл Azure использования. Дополнительные сведения см. в разделе [Регулярно просматривайте на портале детализацию затрат и график расходов](billing-getting-started.md#costs) и статье [Скачивание или просмотр счета на оплату и данных о ежедневном использовании в Azure](billing-download-azure-invoice-daily-usage-date.md). 

### <a name="you-have-reached-the-end-of-your-12-months-free-services"></a>Достигнут конец на 12 месяцев службы бесплатной

Службы бесплатной и количества срок действия истекает в конце 12 месяцев. Чтобы узнать дату истечения срока действия на портале Azure.

1.  Войдите на [портал Azure](http://portal.azure.com).

2.  В нижней части левой области навигации выберите **Дополнительные службы**.

3.  Выберите **Подписки**.

4.  Выберите подписку, созданную при регистрации для получения бесплатной учетной записи.

5.  Прокрутите вниз до сетки бесплатной службы поиска. Щелкните подсказку, расположенных в верхней части левой части сетки.

![Снимок экрана, показывающий, где находится дата окончания срока действия преимущества бесплатной учетной записи](./media/billing-avoid-charges-free-account/freeaccount-benefits-expiration-date.png)


После бесплатной службы и количества истекшим сроком действия, Azure взимает с вас ставки оплаты по мере использования для любой службы, которую вы используете. Портал Azure можно использовать для удаления ресурсов неиспользуемых служб. Если вы не собираетесь использовать какие-либо службы Azure, можно [отказаться от подписки](billing-how-to-cancel-azure-subscription.md).
