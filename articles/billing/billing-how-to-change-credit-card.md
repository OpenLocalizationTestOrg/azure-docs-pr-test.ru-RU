# <a name="change-the-credit-card-used-to-pay-for-an-azure-subscription"></a>Изменение кредитной карты, используемой для оплаты подписки Azure
Вы можете добавить, изменить или удалить кредитную карту, связанную с учетной записью Azure, при входе в [Центр управления учетной записью Azure](https://account.windowsazure.com/Subscriptions) в качестве [администратора учетной записи](billing-subscription-transfer.md#whoisaa). 
 
<a id="addcard"></a>
## <a name="add-a-credit-card-as-a-payment-method"></a>Добавление кредитной карты в качестве метода оплаты

1. Войдите в [Центр управления учетной записью Azure](https://account.windowsazure.com/Subscriptions) в качестве администратора учетной записи.
2. Выберите подписку.
3. В правой части страницы выберите **Manage payment methods**(Управление методами оплаты).
4. Выберите "+" для добавления кредитной карты.
5. Выберите тип кредитной карты и введите необходимые сведения.
6. Щелкните **Далее**. 

Если карта действительна, она будет добавлена в список действительных способов оплаты.

Если после добавления кредитной карты появится ошибка, см. статью [Банковская или кредитная карта отклонена при регистрации в Azure](billing-credit-card-fails-during-azure-sign-up.md).

## <a name="edit-payment-information-for-an-existing-credit-card"></a>Изменение платежной информации для существующей кредитной карты
  Если существующая кредитная карта обновлена, а ее номер остается прежним, обновите данные такой карты, например дату окончания срока действия. Если номер кредитной карты изменен из-за потери, кражи или истечения срока ее действия, следуйте указаниям в разделе [Добавление кредитной карты в качестве метода оплаты](#addcard). 

1. Войдите в [Центр управления учетной записью Azure](https://account.windowsazure.com/Subscriptions) в качестве администратора учетной записи.
2. Выберите подписку, связанную с кредитной картой.</br> ![Снимок экрана, на котором показан выбранный пример подписки.](./media/billing-how-to-change-credit-card/selectsub.png)
3. Выберите **Управление методами оплаты**.</br> ![Снимок экрана, на котором показан выбранный параметр "Управление методами оплаты".](./media/billing-how-to-change-credit-card/changesub_new.png)
4. Выберите метод оплаты.
5. Рядом с номером кредитной карты выберите **Изменить**.</br> ![Снимок экрана, на котором показан параметр "Изменить" рядом с методом оплаты.](./media/billing-how-to-change-credit-card/editcard_new.png)
6. Убедитесь, что указан правильный тип и номер кредитной карты.
7. Щелкните **Далее**.

## <a name="change-the-credit-card-that-pays-your-azure-bill"></a>Изменение кредитной карты, используемой для оплаты счета Azure

1. Войдите в [Центр управления учетной записью Azure](https://account.windowsazure.com/Subscriptions) в качестве администратора учетной записи.
2. Выберите подписку.
3. В правой части страницы выберите **Manage payment methods**(Управление методами оплаты).
4. Щелкните **Использовать замену** и выберите кредитную карту, которую вы хотите использовать.

## <a name="remove-a-credit-card-from-the-account"></a>Удаление кредитной карты из учетной записи
1. Войдите в [Центр управления учетной записью Azure](https://account.windowsazure.com/Subscriptions) в качестве администратора учетной записи.
2. Выберите подписку.
3. В правой части страницы выберите **Manage payment methods**(Управление методами оплаты).
4. Щелкните **Удалить** для кредитной карты, которую требуется удалить.

Если кредитная карта связана с другими активными подписками Microsoft, ее нельзя удалять из учетной записи Azure. Удалите кредитную карту из всех активных подписок Microsoft.

##  <a name="how-to-make-payments"></a>Выполнение оплаты

* Если в качестве метода оплаты вы указали кредитную или платежную карту, оплата производится автоматически.
* Если вы используете [счета](https://azure.microsoft.com/pricing/invoicing/), отправьте платеж в местоположение, указанное внизу счета.
