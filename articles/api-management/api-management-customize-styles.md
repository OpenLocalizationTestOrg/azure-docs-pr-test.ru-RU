![Панель инструментов настройки][api-management-customization-toolbar]

<span data-ttu-id="0c52e-119">Для данного примера щелкните параметр **Select an element on the page** (Выбрать элемент на странице).</span><span class="sxs-lookup"><span data-stu-id="0c52e-119">Click the **Select an element on the page** option for this example.</span></span>  <span data-ttu-id="0c52e-120">Теперь элементы будут выделяться при наведении на них указателя мыши.
