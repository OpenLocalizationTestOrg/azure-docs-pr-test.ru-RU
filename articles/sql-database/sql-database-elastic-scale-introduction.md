<span data-ttu-id="d7de3-170">Другие сценарии объединяют несколько клиентов вместе в базах данных вместо изолирования их в отдельных базах данных.</span><span class="sxs-lookup"><span data-stu-id="d7de3-170">Others scenarios pack multiple tenants together into databases, rather than isolating them into separate databases.</span></span> <span data-ttu-id="d7de3-171">Это типичный **шаблон сегментирования для нескольких клиентов**. ### <a name="move-data-from-multiple-to-single-tenancy-databases"></a><span data-ttu-id="d7de3-177">Перемещение данных из многопользовательских баз данных в однопользовательскую базу данных</span><span class="sxs-lookup"><span data-stu-id="d7de3-177">Move data from multiple to single-tenancy databases</span></span>
