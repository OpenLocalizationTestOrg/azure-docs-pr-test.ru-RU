# <a name="public-data-sets-for-testing-and-prototyping"></a>Общедоступные наборы данных для тестирования и создания прототипов

Просмотрите список общедоступных наборов данных, которые можно использовать для тестирования или создания прототипов служб для хранения, аналитики и аналитических решений.

## <a name="us-government-and-agency-data"></a>Данные учреждений и правительства США

| Источник данных | О данных | О файлах |
|---|---|---|
| [Данные правительства США](https://www.census.gov/data.html) | Более 190 000 наборов данных, содержащих сведения о сельском хозяйстве, климате, потребителях, экосистемах, образовании, энергии, финансах, сфере здравоохранения, органах местного самоуправления, производстве, морской и общественной безопасности, а также о науке и исследованиях в США. | Файлы различных размеров и форматов, включая HTML, XML, CSV, JSON, Excel и многие другие. Вы можете фильтровать доступные наборы данных по формату файла. |
| [Данные переписи населения США](https://www.census.gov/data.html) | Статистические данные о населении США. | Наборы данных доступны в различных форматах. |
| [Геонаучные данные от НАСА](https://earthdata.nasa.gov/) | Более 32 000 наборов данных, содержащих сведения о сельском хозяйстве, атмосфере, биосфере, климате, криосфере, человеческих измерениях, гидросфере, поверхности земли, океанах, взаимодействии Солнца и Земли и многое другое. | Наборы данных доступны в различных форматах. |
| [Задержки рейсов авиакомпаний и другие данные о транспортировках](https://www.transtats.bts.gov/OT_Delay/OT_DelayCause1.asp) | "Бюро статистики Министерства транспорта США отслеживает данные о соблюдении графиков внутренних рейсов больших авиакомпаний. Сводные данные о числе рейсов по графику, задержанных, отмененных и измененных рейсов отображаются в сводных таблицах, размещенных на этом веб-сайте". | Файлы доступны в формате CSV. |
| [Транспортные происшествия со смертельным исходом. Система отчетности по анализу смертности в США (FARS)](http://www.nhtsa.gov/FARS) | "FARS — это система всеобщей переписи населения, ежегодно предоставляющая Национальной администрации безопасности дорожного движения (NHTSA), конгрессу США и американской общественности годовые данные о смертельных травмах, полученных в результате дорожно-транспортных происшествий". | "Создайте собственные данные о смертности в сети с помощью системы запросов FARS или скачайте все данные FARS с 1975 года по настоящее время на сайте FTP". |
| [Данные о токсичности химических веществ. Данные EPA ForeCaster о токсичности (ToxCast™)](https://www.epa.gov/chemical-research/toxicity-forecaster-toxcasttm-data) | "Самые актуальные общедоступные данные EPA с высокой пропускной способностью о токсичности тысяч химических веществ. Эти данные создаются в рамках научно-исследовательской работы EPA ToxCast". | Наборы данных доступны в различных форматах, включая электронные таблицы, пакеты R и файлы базы данных MySQL. |
| [Данные о токсичности химических веществ. Национальный институт здравоохранения. Tox21 Data Challenge 2014](https://tripod.nih.gov/tox21/challenge/) | "Tox21 Data Challenge 2014 предназначен для того, чтобы помочь ученым понять потенциальный риск химических веществ, тестируемых в рамках инициативы "Токсикология в XXI веке" для нарушения биологических путей, которые могут привести к токсическому воздействию". | Наборы данных доступны в форматах SMILES и SDF. В этих наборах содержатся полученные в результате анализа данные о действиях и химические структуры в коллекции Tox21, насчитывающей примерно 10 000 веществ (Tox21 10 тыс.). |
| [Данные о геномах и биотехнологии. Национальный центр биотехнологической информации](https://www.ncbi.nlm.nih.gov/guide/data-software/) | Несколько наборов данных, содержащих сведения о генах, геномах и белках. | Наборы данных доступны в текстовом формате, формате XML, BLAST и других. Доступно приложение BLAST. |
