# <a name="azure-app-service-security"></a>Безопасность службы приложений Azure
Службы приложений Azure соответствуют основным отраслевым стандартам безопасности и надежности и служат безопасной платформой для наших клиентов. Использовать функции защиты входящего и исходящего трафика вашего приложения можно несколькими простыми способами. Кроме того, мы помогаем клиентам защищать код приложений, используя внешние функциональные возможности для проверки приложений на наличие слабых мест.

[!INCLUDE [app-service-blueprint-security](../../includes/app-service-blueprint-security.md)]

Служба приложений Azure использует ту же антивредоносную программу, что и облачные службы Azure и виртуальные машины. Дополнительные сведения об этой программе см. в статье [Антивредоносное ПО Майкрософт для облачных служб и виртуальных машин Azure](../security/azure-security-antimalware.md).
