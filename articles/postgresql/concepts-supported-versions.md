## <a name="managing-updates-and-upgrades"></a>Управление обновлениями
База данных Azure для PostgreSQL автоматически управляет установкой исправлений для обновления дополнительного номера версии. Пока служба доступна в общедоступной предварительной версии, обновление основного номера версии не поддерживается. Например обновление из tooPostgreSQL PostgreSQL 9,5 9,6 не поддерживается.
