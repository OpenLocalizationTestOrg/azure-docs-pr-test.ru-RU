# <a name="get-started-with-hello-box-connector"></a>Начало работы с соединителем поле hello
Подключите панель инструментов и создание файлов, удалите файлы и многое другое. С помощью Box можно:

* Создать поток вашего бизнеса, на основе hello данных, получаемых из поля. 
* использовать триггеры при создании или обновлении файла;
* использовать действия по копированию, удалению файлов и т. д. Эти действия получить ответ и внесите hello выходные данные для других действий. Например, при изменении файла в Box можно отправить измененный файл по электронной почте с помощью Office 365.

Для начала можно создать приложение логики, как описано [здесь](../logic-apps/logic-apps-create-a-logic-app.md).
