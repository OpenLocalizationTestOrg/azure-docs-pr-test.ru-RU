<span data-ttu-id="cd307-133">Вы можете использовать проекты, чтобы группировать приложения и предоставлять другим пользователям разрешения на доступ к ним.</span><span class="sxs-lookup"><span data-stu-id="cd307-133">You can use projects to group your applications and give permissions to other users to access your applications.</span></span> <span data-ttu-id="cd307-134">Разрешения другим пользователям предоставляются посредством указания адреса электронной почты.</span><span class="sxs-lookup"><span data-stu-id="cd307-134">You give permissions to other users by providing email address.</span></span> <span data-ttu-id="cd307-135">Кнопка **Новый проект** позволяет создать новый проект, указав только "имя" и "описание" для него.</span><span class="sxs-lookup"><span data-stu-id="cd307-135">The **New project** button allows you to create a new project by only entering a "name" and a "description" of your new project.</span></span> <span data-ttu-id="cd307-136">После создания вы можете щелкнуть имя проекта, чтобы изменить его имя и описание, а также выбрать все приложения, которые нужно включить в этот проект.</span><span class="sxs-lookup"><span data-stu-id="cd307-136">Once a project is created, you can click on the project name to edit the name and description of your product and to select all the applications you want to see in this project.</span></span>

* <span data-ttu-id="cd307-146">**Администратор**— это пользователь, который может делать все то же, что и разработчик, а также управлять пользователями.</span><span class="sxs-lookup"><span data-stu-id="cd307-146">**Administrator**: An Administrator is a User who can do everything a Developer can do, as well as manage Users.</span></span> <span data-ttu-id="cd307-147">Он может приглашать новых пользователей присоединиться к проекту, менять их роли и информацию о проекте.</span><span class="sxs-lookup"><span data-stu-id="cd307-147">An Administrator can invite users to join a project, can change user roles, and can change project's information.</span></span> <span data-ttu-id="cd307-148">Для этой роли также можно задать разрешения на уровне приложения в меню «Параметры».</span><span class="sxs-lookup"><span data-stu-id="cd307-148">Application level permissions can also be set in “settings”.</span></span>
