## <a name="storage-cost"></a>Стоимость хранения
Оболочки облака создает изображение по умолчанию 5 ГБ в toopersist общего ресурса в файл каталога $HOME. За использование общих папок Azure взимается обычная плата.

Дополнительные сведения о затратах на файлы Azure см. [здесь](https://azure.microsoft.com/pricing/details/storage/files/).
