# <a name="what-is-microsoft-genomics"></a>Что такое Microsoft Genomics?
Геномика Корпорация Майкрософт предлагает облачной реализации Aligner Рогова Burrows (BWA) и набор средств анализа Genome (GATK) для дополнительного анализа. Служба, сертифицированным по стандарту ISO и нормативные требования закона HIPAA и обеспечивает предсказуемость цены для потребностей genome виртуализации. Узнайте, как можно использовать службу Microsoft Геномика и интегрировать с нашей API, считывая наши примеры использования, учебники и документацию.

## <a name="support-your-most-demanding-sequencing-needs"></a>Обеспечение потребностей, связанных с секвенированием
Вместо управления собственными центрами обработки данных воспользуйтесь преимуществами масштабирования и опыта Майкрософт, позволяющими выполнять рабочие нагрузки с эксабайтами данных. Наш хранилища данных облачных служб BWA-GATK высокой concordant с Institute широкой лучшие рекомендации конвейера.


## <a name="keep-your-business-running"></a>Развитие вашего бизнеса
Геномика Корпорация Майкрософт предлагает доступность 99,99% соглашение уровня службы (SLA) для получения запросов рабочего процесса. Дополнительные сведения о Microsoft Геномика [соглашения об уровне ОБСЛУЖИВАНИЯ](https://azure.microsoft.com/support/legal/sla/genomics/v1_0/).


## <a name="secure-your-data"></a>Защита данных
Служба Microsoft Геномика является ISO 27001, ISO 27018 и ISO 9001 сертифицированы и совместимы с нормам HIPAA. Посетите [центр управления безопасностью Azure](https://www.microsoft.com/TrustCenter/Security/default.aspx), чтобы посмотреть сведения о безопасности платформы Azure.
