# <a name="add-an-internal-support-statement-to-a-lab-in-azure-devtest-labs"></a>Добавьте оператор внутренней поддержки лаборатории в Azure DevTest Labs

Azure DevTest Labs позволяет настраивать лаборатории с помощью инструкции внутренней поддержки, которая предоставляет пользователям сведения о лаборатории. Например можно предоставить контактные данные, чтобы пользователь знал о подключении к внутренней поддержки для получения справки по устранения неполадок или доступ к ресурсам в лаборатории. Можно также приведены ссылки на внутренние веб-сайты или часто задаваемые вопросы, имеющие доступ к вашей команды перед обращением в службу поддержки.

Оператор внутренней поддержки предназначен для позволяют помещать сведения о лаборатории, не изменяет обычно слишком часто. Для уведомления пользователей о лаборатории информации, которая более временными — например, последние обновления политик лаборатории — в разделе [разместить объявление в лаборатории](devtest-lab-announcements.md).

Кроме того, можно легко отключить или изменить поддержке после его уже не применима.

## <a name="steps-to-add-a-support-statement-to-an-existing-lab"></a>Шаги для добавления к существующей лаборатории поддержке

1. Войдите на [портале Azure](http://go.microsoft.com/fwlink/p/?LinkID=525040).
1. При необходимости щелкните **Все службы**, а затем выберите в списке **DevTest Labs**. Возможно, лаборатория уже отображается на панели мониторинга в разделе **Все ресурсы**.
1. В списке labs выберите лаборатории, в которой требуется добавить поддержке.  
1. В области лаборатории **Обзор** выберите **Configuration and policies** (Конфигурация и политики).  

    ![Кнопка Configuration and Policies (Конфигурация и политики)](./media/devtest-lab-internal-support-message/devtestlab-config-and-policies.png)

1. В левой части экрана, в разделе **параметры**выберите **внутренней поддержки**.

    ![Кнопка внутренней поддержки](./media/devtest-lab-internal-support-message/devtestlab-internal-support.png)

1. Чтобы создать сообщение внутренней поддержки для пользователей в этой лаборатории, присвоено значение Enabled **Да**.

1. В **сообщение о поддержке** введите инструкцию внутренней поддержки, который требуется предоставить пользователям лаборатории. Сообщение о поддержке принимает разметки. При вводе текста сообщения, можно просмотреть **предварительного просмотра** области в нижней части экрана, чтобы увидеть, как будет отображаться сообщение пользователям.

    ![Экран внутренней поддержки для создания сообщения.](./media/devtest-lab-internal-support-message/devtestlab-add-support-statement.png)


1. Выберите **Сохранить** после заявление о поддержке готов к публикации.

Если вы хотите больше не показывать это сообщение поддержки пользователям лаборатории, возвращают **внутренней поддержки** и задайте **включено** для **нет**.

## <a name="steps-for-users-to-view-the-support-message"></a>Шаги для пользователей просмотреть сообщение о поддержке

1. На [портале Azure](http://go.microsoft.com/fwlink/p/?LinkID=525040) выберите лабораторию.

1. В лаборатории **Обзор** выберите **внутренней поддержки**.  

    ![Кнопка внутренней поддержки](./media/devtest-lab-internal-support-message/devtestlab-internal-support.png)


1. Если отправляется сообщение поддержки, пользователь может просмотреть его из области внутренней поддержки.

    ![Опубликовано сообщение поддержки внутренней поддержки панелью](./media/devtest-lab-internal-support-message/devtestlab-view-suport-statement.png)
