# <a name="deploy-the-azure-iot-device-simulation-solution"></a>Развертывание решения моделирования устройств IoT Azure

Этого учебника показано, как подготовить решение имитации устройства. Решение развертывается на сайте azureiotsuite.com.

Из этого руководства вы узнаете, как выполнять такие задачи:

> [!div class="checklist"]
> * Настройка решения моделирования устройства
> * Развертывание решения моделирования устройства
> * Вход в решение имитации устройства

## <a name="prerequisites"></a>Технические условия

Для работы с этим руководством требуется активная подписка Azure.

Если ее нет, можно создать бесплатную пробную учетную запись всего за несколько минут. Дополнительные сведения см. в разделе [Бесплатная пробная версия Azure](http://azure.microsoft.com/pricing/free-trial/).

## <a name="deploy-the-solution"></a>Развертывание решения

Прежде чем развертывать решения с вашей подпиской Azure, необходимо выбрать некоторые параметры конфигурации:

1. Войдите в [azureiotsuite.com](https://www.azureiotsuite.com) Azure с помощью учетных данных учетной записи и нажмите кнопку  **+**  для создания нового решения:

    ![Создание решения](media/iot-suite-device-simulation-deploy/createnewsolution.png)

1. Нажмите кнопку **выберите** на **моделирование устройства** плитки:

    ![Выберите устройство моделирования](media/iot-suite-device-simulation-deploy/select.png)

1. На **создать имитации устройствами** введите **имя решения** решения моделирования устройства.

1. Выберите **регион** и **подписку**, которые вы хотите использовать для подготовки решения.

1. Укажите, если центр IoT развернуты вместе с решением имитации устройства. Если установить этот флажок, новый центр IoT развернута в подписке. Вне зависимости от выбора можно всегда указывать вашей модели в любой центр IoT.

1. Щелкните **Создать решение** , чтобы начать процесс подготовки. Этот процесс обычно занимает несколько минут:

    ![Сведения о решении имитации устройства](media/iot-suite-device-simulation-deploy/createsolution.png)

## <a name="sign-in-to-the-solution"></a>Вход в решение

После завершения процесса подготовки сможете войти в решение имитации устройства.

1. На **подготовить решения** щелкните **запуска** в новом решении имитации устройства:

    ![Выбор нового решения](media/iot-suite-device-simulation-deploy/newsolution.png)

1. Можно просмотреть сведения о решении имитации устройства панели, который отображается. Выберите **решений мониторинга** для подключения к решению имитации устройства.

    > [!NOTE]
    > Решения моделирования устройства можно удалить из этой панели при завершении работы с ним.

    ![Панель решения](media/iot-suite-device-simulation-deploy/properties.png)

1. Панели мониторинга симуляции устройства решение отображается в браузере.

## <a name="next-steps"></a>Дальнейшие действия

Из этого руководства вы узнали, как выполнить следующие задачи:

> [!div class="checklist"]
> * Настройка решения
> * Развертывание решения
> * Вход в решение

Теперь, когда развертывания решения моделирования устройства, следующим шагом является [Изучите возможности решения моделирования устройства](./iot-suite-device-simulation-explore.md).
