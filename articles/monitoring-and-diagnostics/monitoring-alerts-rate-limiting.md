# <a name="rate-limiting-for-sms-messages-emails-azure-app-push-notifications-and-webhook-posts"></a>Интенсивность ограничение для SMS-сообщений, сообщений электронной почты, push-уведомлений приложения Azure и веб-перехватчик сообщений
Ограничение скорости является приостановку уведомления происходит, когда слишком много уведомления отправляются нужный номер телефона, адрес электронной почты или устройстве. Ограничение скорости гарантирует управляемость и действенность оповещений.

Ограничение частоты пороговые значения являются:

 - **SMS**: не более 1 SMS каждые 5 минут.
 - **Электронная почта**: 100 сообщений в час.
 - **Azure Push-уведомлений приложения**: отсутствии курса ограничение для push-уведомлений.
 - **Веб-перехватчиков**: отсутствии курса ограничение для веб-привязок.

## <a name="rate-limit-rules"></a>Правила ограничения частоты
- Нужный номер телефона или адрес электронной почты ограничивается, если частота превысит пороговое значение.
- Номер телефона или адрес электронной почты может быть частью групп действий в нескольких подписках. Ограничение частоты применяется ко всем подпискам по достижении порогового значения, даже если сообщения отправляются из нескольких подписок.  
- При ограниченной по скорости, адрес электронной почты дополнительных уведомление отправляется для взаимодействия ограничения скорости. Уведомление сообщает также, когда истекает срок действия ограничения частоты.
